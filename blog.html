4. What is a callback function and why will you use it?

Call BAck function : callback function is function that waiting for another function executing before executing itself. a function like a object in javascript .for this ,function can take other function as a arguments and also can return by another function. when a function take or call another function as a argument or return a function as result, that function called a higher-order function. Any function that is passed as an arguments is called a callback function
